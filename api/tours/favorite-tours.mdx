## Overview

The Favorite Tours API allows tourists to save tours to their favorites list for easy access. This creates a personalized wishlist of tours they're interested in.

## Base URL

```
/api/favorite-tours
```

## Get All Favorite Tours

Retrieve all favorite tours across all users (typically used for admin purposes).

<CodeGroup>
```bash cURL
curl -X GET "https://api.kinconecta.com/api/favorite-tours" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.kinconecta.com/api/favorite-tours', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const favorites = await response.json();
```
</CodeGroup>

### Response

<ResponseField name="favorites" type="Array<FavoriteTour>">
  List of all favorite tour records
  
  <Expandable title="FavoriteTour properties">
    <ResponseField name="favoriteId" type="Long">
      Unique identifier for the favorite record
    </ResponseField>
    
    <ResponseField name="touristId" type="Long">
      ID of the tourist who favorited the tour
    </ResponseField>
    
    <ResponseField name="tourId" type="Long">
      ID of the favorited tour
    </ResponseField>
    
    <ResponseField name="createdAt" type="DateTime">
      Timestamp when the tour was favorited
    </ResponseField>
  </Expandable>
</ResponseField>

## Get Favorite by ID

Retrieve a specific favorite tour record.

<CodeGroup>
```bash cURL
curl -X GET "https://api.kinconecta.com/api/favorite-tours/{favoriteId}" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const favoriteId = 123;
const response = await fetch(`https://api.kinconecta.com/api/favorite-tours/${favoriteId}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const favorite = await response.json();
```
</CodeGroup>

### Path Parameters

<ParamField path="favoriteId" type="Long" required>
  The ID of the favorite tour record
</ParamField>

### Response

Returns a single FavoriteTour object.

## Add Tour to Favorites

Add a tour to a tourist's favorites list.

<CodeGroup>
```bash cURL
curl -X POST "https://api.kinconecta.com/api/favorite-tours" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "touristId": 456,
    "tourId": 789
  }'
```

```javascript JavaScript
const response = await fetch('https://api.kinconecta.com/api/favorite-tours', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    touristId: 456,
    tourId: 789
  })
});
const favorite = await response.json();
```
</CodeGroup>

### Request Body

<ParamField body="touristId" type="Long" required>
  ID of the tourist adding the favorite
</ParamField>

<ParamField body="tourId" type="Long" required>
  ID of the tour to add to favorites
</ParamField>

### Response

Returns the created FavoriteTour object with generated ID and timestamp.

```json
{
  "favoriteId": 123,
  "touristId": 456,
  "tourId": 789,
  "createdAt": "2024-03-10T15:30:00Z"
}
```

## Remove Tour from Favorites

Remove a tour from a tourist's favorites list.

<CodeGroup>
```bash cURL
curl -X DELETE "https://api.kinconecta.com/api/favorite-tours/{favoriteId}" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const favoriteId = 123;
const response = await fetch(`https://api.kinconecta.com/api/favorite-tours/${favoriteId}`, {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
```
</CodeGroup>

### Path Parameters

<ParamField path="favoriteId" type="Long" required>
  The ID of the favorite record to delete
</ParamField>

### Response

Returns the deleted FavoriteTour object.

<Note>
The favorite record is deleted from the database. The tourist can add the tour back to favorites at any time.
</Note>
